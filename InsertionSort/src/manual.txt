Для запуска нужно в командной строке ввести:
1) Перейти в папку с файлом.
2) Набрать java -jar имя файла, указать аргументы для запуска:
   Имя входного файла (например in.txt)
   Имя выходного файла (например out. txt)
   Тип сортируемых данных(-i целые числа -s строки)
   Вид сортировки (-а по возрастанию -d по убыванию)

   Например:
   java -jar InsertionSort.jar in.txt out.txt -i -a сортировка целых чисел по возрастанию.
